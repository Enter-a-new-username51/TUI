## 💻 Jak uruchomić (Installation)

Instrukcja krok po kroku, jak odpalić Twój projekt na innym komputerze:

1. Sklonuj repozytorium:
```bash
git clone https://github.com/Enter-a-new-username51/TUI
```
2. Zainstaluj Microsoft build tools
3. Uruchom x64 Native Tools Command Prompt for VS
4. Wpisz `cd [SCIEZKA_DO_FOLDERU_TEGO_REPOZYTORIUM_NA_TWOIM_KOMPUTERZE]` i naciśnij enter.
5. Uruchom nmake i czekaj aż program się skompiluje.
6. Uruchom Test_For_TUI.exe
